* [Oh My](https://ohmyz.sh/) [Zsh](https://en.wikipedia.org/wiki/Z_shell)

* [JetBrains Mono](https://www.jetbrains.com/lp/mono/)
* [Fzf](https://github.com/junegunn/fzf)
* [Compton](https://github.com/chjj/compton)
